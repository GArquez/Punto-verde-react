# Punto Verde

> Sitio ecommerce que permite simular ordenes de compra guardandolas en Firebase.

## Para correr el proyecto en tu local

```bash
$ git clone https:/https://github.com/GArquez/Punto-verde-react
$ cd punto-verde
$ npm install
$ npm start
```
